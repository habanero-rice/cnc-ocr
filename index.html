Concurrent Collections on OCR

CnC-OCR is a high-level, graph-oriented programming paradigm that is built on top of the Open Community Runtime (OCR). The full CnC-OCR toolchain is distributed as part of the open source OCR project.

Please visit our GitHub wiki for setup instructions and further resources on using CnC-OCR.
